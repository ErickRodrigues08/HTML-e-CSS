float css


A propriedade CSS float permite que um elemento saia do fluxo normal da página e "flutue" para a esquerda ou direita, permitindo que o conteúdo adjacente (como texto) se posicione ao seu redor.


Valores mais comuns

  • float: left;: O elemento flutua para a esquerda de seu contêiner.
  • float: right;: O elemento flutua para a direita de seu contêiner.
  • float: none;: O elemento não flutua (valor padrão).
  • float: inherit;: Herda o valor da propriedade do elemento pai.
